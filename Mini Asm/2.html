Pls read the rules before you start the game.

1. The system chooses a random number between 1 and 100.

2. The guesser should guess a number and type it in the space.

3. If the guess is too high, the system will say "too high". Guessers can use previous guesses to help them make the next guess.

4. If the guess is too low, the system will say "too low". Guessers continue to guess and get answers until they guess right.

5. Finally, the system will calculate the number of guesses required to get the random number.
